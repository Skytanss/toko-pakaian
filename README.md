# Website Toko Pakaian Online - FashionStore

Website toko pakaian online responsif yang dibangun dengan HTML, CSS, dan JavaScript.

## Fitur Utama

1. **Halaman Utama** 
   - Hero section dengan promosi
   - Kategori produk
   - Produk unggulan
   - Fitur toko

2. **Halaman Produk**
   - Grid produk dengan filter kategori
   - Fitur pencarian
   - Pagination (jika diperlukan)

3. **Halaman Detail Produk**
   - Detail lengkap produk
   - Gallery gambar
   - Pilihan ukuran/warna
   - Quantity selector
   - Produk terkait

4. **Keranjang Belanja**
   - Tambah/hapus produk
   - Update quantity
   - Ringkasan belanja
   - Perhitungan otomatis

5. **Checkout**
   - Form pengiriman
   - Pilihan metode pembayaran
   - Ringkasan pesanan
   - Konfirmasi pesanan

6. **Halaman Informasi**
   - Tentang Kami
   - Kontak dengan form
   - FAQ

7. **Fitur Responsif**
   - Mobile-first design
   - Navigation menu mobile
   - Layout adaptif untuk semua ukuran layar
